*** HTO library quantification ***

- Quantify HTO libraries using the Tag information from BioLegend TotalSeq-A antibodies
  and the CITE-seq-Count script in python (https://hoohm.github.io/CITE-seq-Count/)
  
  Using version: CITE-seq-Count v1.4.5
  
	% tags.csv: csv file with TotalSeq-A antibody barcode information for each sample
	% get_HTO_Counts.sh: bash script with parameters for CITE-seq-Count
	% Output: folder containing the compressed output folders from quantification
